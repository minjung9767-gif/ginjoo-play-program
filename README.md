# 아기 놀이터 🎈 (ginjoo-play-program)

22개월 아기를 위한 카메라 기반 웹 놀이입니다. 설치 없이 브라우저에서 바로 실행돼요.
아기 영상은 기기 밖으로 나가지 않고 브라우저 안에서만 처리됩니다. (개인용)

## 지금 만들어진 놀이

### 🪞 매직 거울 (완성)
- 카메라로 얼굴을 비추면 **동물 귀 + 안경/선글라스 + 모자**가 척척 붙어요.
- 고개를 움직이면 스티커가 따라 움직이고, **몇 초마다 스타일이 자동으로 바뀝니다.**
- **화면을 톡 누르면** 그 자리에 ⭐💖🫧 가 팡 나타나고 경쾌한 소리가 나요.
- 밝고 통통 튀는 배경 음악이 흐릅니다. (직접 만든 멜로디 — 저작권 걱정 없음)

> ✨ 움직임 마법, 🐻 까꿍 동물은 "준비 중" — 다음 단계에서 추가 예정입니다.

## 실행 방법

카메라는 보안상 **그냥 파일을 더블클릭해서 열면 동작하지 않아요.** 아래 둘 중 하나로 실행하세요.

### 방법 1) 간단한 로컬 서버 (권장)
이 폴더에서 아래 명령을 실행한 뒤, 브라우저에서 `http://localhost:8000` 으로 접속:

```bash
python3 -m http.server 8000
```
(파이썬이 없으면 `npx serve` 도 가능)

그다음 브라우저에서 **🪞 매직 거울** 버튼을 누르고, 카메라 권한을 **허용**하면 시작돼요.

### 부모용 컨트롤
화면 오른쪽 위 모서리에 작게 있어요 (아기가 잘못 누르지 않도록):
- 🔊 / 🔇 : 소리 켜기/끄기
- 🏠 : 시작 화면으로 돌아가기 (카메라도 꺼짐)

## 사용 기기
- **삼성 노트북(가로, 터치)** 기준으로 만들었고, 아이패드·맥북 등 다른 기기에서도 화면 크기에 맞춰 자동으로 조정됩니다.
- 권장 브라우저: 크롬(Chrome), 엣지(Edge).

## 기술 메모 (개발용)
- 순수 HTML/CSS/JS, 빌드 과정 없음.
- 얼굴 인식: Google MediaPipe Tasks Vision (FaceLandmarker) — CDN에서 로드.
- 스티커는 별도 이미지 파일 없이 **이모지 + 캔버스 그림**으로 구현(귀는 직접 그림).
- 배경 음악/효과음은 Web Audio API로 합성 (음원 파일 없음).
- 인터넷 연결 필요(첫 실행 시 인식 모델 다운로드, 이후 캐시됨).

## 파일 구조
```
index.html          시작 화면 + 게임 화면
css/style.css       반응형 스타일
js/main.js          화면 전환 / 수명주기 관리
js/camera.js        카메라 시작·정지
js/audio.js         배경 음악 + 효과음
js/games/mirror.js  매직 거울
```
